# Курс PostgreSQL для администраторов и разработчиков (OTUS)

Я прохожу курс "PostgreSQL для администраторов баз данных и разработчиков" на OTUS.

**Ссылка:** [https://otus.ru/lessons/postgresql-dba/](https://otus.ru/lessons/postgresql-dba/)

**Описание:** Курс обеспечивает глубокое освоение PostgreSQL: от основ реляционных БД, SQL, установки и конфигурации до оптимизации запросов (DDL, DML, индексы, партиционирование), администрирования (backup, репликация, Patroni, мониторинг) и работы с большими данными.

**Цели:**
- Настроить кластер PostgreSQL для оптимальной производительности и восстановления от сбоев.
- Диагностировать и решать проблемы с блокировками, deadlocks и производительностью.
- Оптимизировать запросы с использованием индексов, джойнов, статистики и продвинутых типов данных.
- Освоить репликацию, кластеризацию и мониторинг для высокодоступных систем.
- Применить навыки на практике в проекте с интеграцией и тестированием кластера.
